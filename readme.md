# Khan Academy Front-End Project

Welcome to our Khan Academy Front-End Group Project!

This repository contains the code and resources for our front-end group project as part of the Khan Academy course. Our goal was to create a responsive, user-friendly web application while implementing modern HTML, CSS, and JavaScript practices.

## 🌟 Project Overview

This project focuses on building a fully functional, interactive, and visually appealing front-end interface that demonstrates:

- Responsive design for multiple devices  
- Interactive UI components  
- Proper semantic HTML structure  
- Modern CSS styling techniques  
- Dynamic behavior using JavaScript  

This project allowed our group to collaborate, practice front-end best practices, and showcase our coding skills.

## 🛠️ Technologies Used

- HTML5 – Semantic markup and structure  
- CSS3 – Styling, Flexbox, and Grid layout  
- JavaScript (ES6) – DOM manipulation, event handling, and interactivity  
- VS Code – Primary code editor  
- Git & GitHub – Version control and collaboration
